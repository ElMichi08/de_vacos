> reemplazando un proceso 100% manual. **Resultado: ingresos escalaron 5x desde la implementación.**

---

### ¿Qué problema resuelve?

El restaurante gestionaba pedidos, caja e inventario manualmente — errores de despacho frecuentes,
cierre de caja caótico, pagos registrados por WhatsApp. Esta app reemplazó todo ese flujo con un
sistema POS offline-first que funciona sin conexión y sincroniza con la nube cuando hay red.

| Antes | Después |
|-------|---------|
| Pedidos en papel, errores de despacho constantes | POS digital con pantalla de cocina en tiempo real |
| Cobros registrados por WhatsApp | Módulo de caja con soporte de cobros mixtos (hasta 3 métodos por pedido) |
| Sin visibilidad de inventario | Control de stock en tiempo real, alertas de sobreventa |
| Reportes inexistentes | Panel web admin con reportes de ventas y top productos |
| **Capacidad limitada de atención** | **Ingresos 5x desde la implementación** |
